# IPMS (Front-end)

Clone repository by running a command in the command prompt:

`git clone https://github.com/andriusbr/juvare-ipms-fe.git`

Navigate to newly created directory:

`cd juvare-ipms-fe`

Install NPM packages:

`npm install`

Start the project:

`npm start`
